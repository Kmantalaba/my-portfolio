# Klent Mantalaba Portfolio

A modern personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This portfolio is designed to support full-time software engineering applications and attract freelance clients. It includes:

- A responsive hero section
- About, Experience, Projects, Skills, Education, and Contact sections
- A downloadable PDF resume
- Smooth scrolling and subtle animations

## Tech Stack

- HTML
- CSS
- JavaScript
- jsPDF for browser-based PDF resume generation
